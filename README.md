# zhweb-xingkong-resource

## 项目简介

zhweb-xingkong-resource 是一个用于归档和发布多个独立 HTML 页面的资源仓库。本仓库旨在集中存放与“星空”主题相关的网页文件及配套资源，方便浏览、测试或二次开发。

仓库内包含的页面均为独立 HTML 文件，不依赖特定域名或网站环境，可直接在浏览器中打开使用。

## 目录结构

```
zhweb-xingkong-resource/
├── pages/             # 存放 HTML 页面文件
├── assets/            # 页面所需的静态资源（样式、脚本、图片等）
├── docs/              # 相关文档或说明
└── README.md          # 本文件
```

目录结构可能随版本更新调整，请以实际仓库内容为准。

## 页面归档说明

本仓库收录的 HTML 页面均保持原始文件结构，未对内部代码进行修改或适配。部分页面可能引用外部资源（如字体库、图标库），建议在离线环境下使用时自行替换为本地资源。

页面文件命名以内容主题或功能描述为主，不包含具体域名信息。所有页面仅供学习、参考或归档用途。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 进入 `pages/` 目录，直接双击 HTML 文件即可在浏览器中打开。
3. 如需修改样式或脚本，可编辑对应文件。
